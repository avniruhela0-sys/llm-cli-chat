# 🤖 LLM CLI Chat

A powerful command-line chat application that brings Google's Gemini AI directly to your terminal. Ask questions, get instant responses, and have natural conversations with AI – all without leaving your command line.

---

## ✨ Features

- **💬 Interactive Chat Loop** – Continuous conversation until you type `exit`
- **🔒 Secure API Key Storage** – Uses `.env` file to keep your keys safe (never exposed in code)
- **💰 Free to Use** – Leverages Gemini's free tier – no credit card required
- **🚀 Lightweight & Fast** – Minimal dependencies, runs instantly
- **🖥️ Terminal Native** – No browser needed, works right in your terminal
- **🧠 Powered by Gemini** – Access to Google's latest AI models

---

## 📋 Prerequisites

- **Python 3.8+** installed on your system
- **A Gemini API key** (free – get it from [Google AI Studio](https://makersuite.google.com))
- **Git** (for cloning the repository)
